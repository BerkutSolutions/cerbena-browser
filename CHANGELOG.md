# Журнал изменений

## 1.1.1

### Ядро
- Исправлен импорт расширений для профилей `librewolf`: при формировании URL установки теперь используется движок-специфичный путь пакета расширения, поэтому расширения, добавленные для LibreWolf, корректно подхватываются при запуске профиля.


### Профили
- В модальном окне профиля кнопка добавления сертификата в русской локализации переименована с `Добавить сертификат` на `Добавить` для более короткой и единообразной подписи.
- Сценарий `published-updater-e2e` изолирован от рабочей установки пользователя: тестовая установка и проверка обновления выполняются только в временном `sessionRoot`, добавлен защитный запрет на использование реального `LOCALAPPDATA`.
- Очистка после `published-updater-e2e` стала безопаснее: деинсталляция MSI выполняется только для продуктов, чья `InstallLocation` находится внутри текущей временной сессии, чтобы тест не затрагивал продовую установку.
- Улучшена стабильность `published-updater-e2e`: добавлена диагностика и fallback при зависании статуса обновления в `idle`, чтобы тест корректно переключался на резервный путь и не зависал бесконечно.
