# Журнал изменений

## 1.0.13

### Профили

- Добавлено сохранение удалённых встроенных профилей: после удаления `Chromium Default`, `Firefox Default`, `Chromium Private Memory`, `Firefox Private Memory`, `Discord` или `Telegram` launcher больше не создаёт их заново после перезапуска и обновления.
- В таблицу профилей добавлена сортировка по нажатию на заголовки `Имя`, `Теги` и `Заметка`.
- Исправлена синхронизация назначенных сертификатов между разделом `Безопасность` и редактором профиля: профильные сертификаты теперь отображаются при редактировании и сохраняют назначение в общем хранилище.

### Настройки

- В настройки добавлен чекбокс запуска вместе с системой.
- Для автозапуска добавлен выбор профиля по умолчанию через выпадающий список с чекбоксами в стиле выбора профилей для сертификатов.
- В модальном окне предложения свернуть приложение в трей порядок кнопок изменён на `Да`, `Нет`, `Отмена`.

### Wayfern

- Убрана временная расширенная диагностика `Wayfern`: launcher снова запускает движок с рабочим каталогом профиля, не пишет пользователю `wayfern-debug.log`, `wayfern-stdout.log` и `wayfern-stderr.log`, а окно просмотра логов больше не показывает этот внутренний шум.
