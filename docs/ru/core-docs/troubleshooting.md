## Профиль не запускается

- проверьте, существует ли путь к бинарному файлу;
- проверьте операторскую диагностику runtime на ошибки подготовки и запуска.

## Трафик блокируется неожиданно

- откройте `Traffic` и проверьте причину;
- проверьте kill-switch и глобальный `block without VPN`;
- проверьте `selected_services`, blocklists и правила запрета доменов.

## Не стартует маршрутный runtime

- посмотрите, какой backend нужен: `sing-box`, `openvpn`, `amneziawg` или `tor`;
- проверьте, прошла ли фаза управляемой подготовки;
- проверьте ошибки в runtime-логах профиля.

## Есть проблемы с sync

- откройте `Settings > Sync`;
- проверьте текущий статус и доступность endpoint;
- убедитесь, что у профиля есть снимки перед восстановлением.

## Пропали локализации

- выполните `cd ui/desktop && npm run i18n:check`;
- проверьте паритет ключей `ru` и `en`;
- проверьте отсутствие битой кодировки в русских строках.
