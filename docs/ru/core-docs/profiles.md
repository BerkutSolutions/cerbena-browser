Профиль в Cerbena Browser - это отдельный контейнер состояния браузера со своими данными, политиками, расширениями и криптографическим контекстом.

## Что хранит профиль

- имя, описание, теги и engine (`Wayfern` или `Camoufox`);
- стартовую страницу и поисковый провайдер либо строгий одностраничный режим окна приложения;
- настройки `Identity`, `Network`, `DNS`, `Extensions`, `Security`, `Sync`;
- runtime-состояние, след аудита и кэш;
- при необходимости - парольная блокировка и `ephemeral`-режим.

По умолчанию новые профили стартуют с `https://duckduckgo.com`.

Во встроенный набор теперь входят:

- `Chromium Default`
- `Firefox Default`
- `Chromium Private Memory`
- `Firefox Private Memory`
- `Discord` — строгий профиль `Wayfern` в формате окна приложения только для доменов Discord
- `Telegram` — строгий профиль `Wayfern` в формате окна приложения только для доменов Telegram

## Изоляция

- каждый профиль использует собственный каталог `profiles/{profile_uuid}/`;
- данные, кэш и расширения не разделяются между профилями;
- ключи и секреты не должны переиспользоваться между профилями;
- операции очистки, импорта и экспорта всегда выполняются в рамках выбранного профиля.

## Основные операции

- создание и редактирование;
- дублирование, импорт и экспорт;
- запуск и остановка;
- блокировка и разблокировка;
- выборочная очистка и аварийная очистка.

## Что важно в текущей модалке профиля

- разделы расположены слева вертикальным списком;
- `Identity` соблюдает те же правила `Автоматически` / `Ручной`, что и основная страница;
- `DNS` скрывает ручные поля в системном режиме и показывает их в ручном;
- `Extensions` работает по именам расширений, а не по сырым идентификаторам;
- `Security` объединяет доменные правила разрешения и запрета в один раздел с поиском и показывает парольную блокировку как явный поток `пароль + подтверждение`.

## Специальные режимы

- `Password lock`: требует явной разблокировки перед запуском.
- `Ephemeral`: очищает временные данные при закрытии профиля.
- `Private memory profile`: преднастроенный вариант для приватных сессий.
- `Одностраничный`: доступен для профилей `Wayfern`; запускает строгое окно приложения без свободной адресной строки, скрывает выбор поисковика по умолчанию и ограничивает навигацию доменом стартовой страницы.

## Практические советы

- отделяйте рабочие, исследовательские и тестовые профили;
- не смешивайте политику маршрутизации разных сценариев в одном профиле;
- используйте отдельные профили под разные наборы расширений и модели доверия.
