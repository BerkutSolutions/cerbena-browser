Cerbena Browser допускает локальную автоматизацию, но не жертвует ради нее zero-trust-принципами.

## Локальный API

Подходит для автоматизации на рабочем столе и контролируемого открытия ссылок.

## MCP

Подходит для агентных сценариев, где нужен ограниченный набор инструментов для работы с профилями и политиками.

## Обязательные защитные ограничения

- явная область профиля;
- авторизация на стороне backend;
- события аудита;
- отказ при невалидном контексте.
