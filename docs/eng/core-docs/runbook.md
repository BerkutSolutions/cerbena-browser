## Daily cycle

1. Review profile list and state.
2. Confirm route templates and DNS blocklists are current.
3. Update `Identity`, `Extensions`, or `Sync` settings if required.
4. Review `Settings > Links` when the profile is used for external-link handling.
5. Launch the profile and watch `Traffic`.

## Before launching a sensitive profile

- verify the route mode;
- verify kill-switch state;
- verify `selected_services` and domain rules;

## Before wipe or restore

- verify the target profile;
- confirm the profile is not actively running;
- create a snapshot when possible.
